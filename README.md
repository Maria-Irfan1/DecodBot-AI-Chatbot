# DecodBot AI Chatbot (Python Fundamentals Engine)

### 🚀 DecodeLabs Industrial Training — Batch 2026 (Project 1 Milestone)
This repository contains a humanized, rule-based conversational desktop chatbot interface. It is built strictly using core **Programming Fundamentals** rather than complex abstract modules to prioritize pure logical visibility and deterministic safety boundaries.

---

## 🛠️ Architectural Workflow (IPO Model)
The program processes data flow in a structured three-phase pipeline:
1. **Input Sanitization & Normalization:** Converts incoming strings to uniform lowercase, extracts messy trailing/leading spaces, and safely loops past punctuation flags.
2. **Deterministic Processing Engine:** Evaluates incoming user tokens against explicit string array keywords using pure `if-elif-else` control gates with a mandatory fallback shield.
3. **Continuous Visual Output Loop:** Drives a state persistence terminal loop until an explicit departure command sequence is entered.

* **Standard Built-in Imports:** `string`, `datetime`
* **Core Concepts:** Continuous conditional `while` loops, data normalization, membership filtering arrays (`in [...]`), modular functional abstraction.
